# RogoLabs License Policy

## Standard License: Apache-2.0

All RogoLabs repositories use the **Apache License, Version 2.0**.

## Why Apache-2.0

- **Patent grant**: Protects contributors with an explicit patent license — important for security tooling that may touch patented techniques.
- **Commercial-friendly**: Permits commercial use, modification, and distribution with attribution.
- **Attribution required**: Users must include the license and state changes, giving RogoLabs credit.
- **Industry standard**: Widely used by security tooling organizations (CISA, Apache Foundation, Google).

## History

- **2026-04-09**: Standardized all 17 repos to Apache-2.0. Previously a mix of MIT (5), Apache-2.0 (6), GPL-3.0 (1), and unlicensed (5).

## For New Repos

When creating a new RogoLabs repository, select **Apache-2.0** as the license. GitHub's repo creation UI includes it in the license dropdown.
